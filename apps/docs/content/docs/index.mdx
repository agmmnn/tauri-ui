`create-tauri-ui` scaffolds a modern Tauri app by combining the official shadcn/ui CLI with create-tauri-app, then layering in the desktop defaults most projects add later anyway.

## Why tauri-ui

- Upstream-first: no large local template tree to keep in sync.
- Desktop-aware: window behavior, startup flash prevention, scrolling, and selection are handled from the start.
- Practical defaults: batteries are small, composable, and focused on real app setup work.
